HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • An HTML attribute defines additional information about an element. This is always done in the beginning of the element.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • The 'style' element defines the CSS styles associated with the website.
  • The 'base' element defines the base URL for a webpage.
  • The 'link' elementconnects an external resource to the HTML document.
  • The 'meta' element defines metadata such as the character set, description, keywords, author, and viewport.
  • The 'script' element embeds client-side scripts in the HTML document.
  • Elements that are nested inside other elements are called child elements.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • The header appears at the top of the page and usually contains a navigation menu and/or introductory content
  • The main HTML element represents the dominant content of the body of a document.
  • The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

GIT

  • git revert: revert to an earlier version of the code, possibly to fix a bug.
  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch & switches to it.
  • git-cheat-sheet
  • A branch is a copy or version of the code that can be worked on independently in isolation so development changes do not disrupt other team members who are also working on the code and testing their solutions.
  • In a feature branch, we make commits of all the changes we make to the code while working. Next we push, then open a pull request to merge the feature branch into the main branch.
  • git pull origin main: pull down any changes from the main branch, to ensure the local branch is in sync.
